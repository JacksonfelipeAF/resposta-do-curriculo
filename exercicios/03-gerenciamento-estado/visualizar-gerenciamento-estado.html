Quando usar cada um?

Angular Signals

  • Estado local de componente
  • Formulários e UI state
  • Cálculos simples e derivados
  • Componentes independentes

NgRx

  • Estado global da aplicação
  • Dados compartilhados entre componentes
  • Fluxos assíncronos complexos
  • Aplicações enterprise
